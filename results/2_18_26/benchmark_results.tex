\section*{Methodology Notes}

\begin{itemize}
\item \textbf{Confidence intervals}: Wilson score intervals at 95\% confidence level, appropriate for binomial proportions near 0 or 1.
\item \textbf{Matrix Recall}: Each task generates a fresh random matrix with labeled rows/columns. The model must return exact integer values in JSON format. Partial credit is not given.
\item \textbf{Word Instruction Following}: Score is the mean fraction of 30 sequential word-list instructions executed correctly per rollout. Wilson CI applied to the per-rollout mean score.
\item \textbf{API failures}: Several Azure-hosted models (deepseek-r1, deepseek-v3, phi-4-mini, phi-4-mini-reasoning, llama variants, mistral-nemo) experienced persistent rate-limit or authentication errors, resulting in $n \leq 1$. These are excluded from substantive analysis.
\item \textbf{Ollama models}: Run locally with concurrency 1--8 depending on model size. No API cost.
\item \textbf{In-progress runs}: Ollama WIF is still completing for gemma3n-e2b, gemma3n-e4b, gpt-oss-20b, and phi4-mini-reasoning. Results will be updated.
\end{itemize}
